Projeto Game Mata Mosquito

Projeto envolvendo as tecnologias HTML, CSS e JS.

O jogo tem como objetivo, matar todas as moscas que irão aparecer aleatóriamente pela tela. Existe um intervalo de tempo máximo que você pode demorar para clicar a mosca seguinte. Caso o tempo seja excedido, o jogador perde.

Vale lembrar que existem dificuldades diferentes para a partida e quanto maior o grau de complexidade, menor o tempo de intervalo
